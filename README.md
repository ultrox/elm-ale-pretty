## Pretty Error Message

> Makes Elm error message pretty in :ALEDetail


## Install

`Plug 'ultrox/elm-ale-pretty'`


### To disable this plug-in:
` let g:disable_elm_ale_pretty = 1 to your .vimrc or vim.init`
